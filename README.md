# ReAmBaP - Remove Ambiguous Base Pairs

Remove the ambiguous base pairs from your FASTA file. <br>
Ideal for tools like Snippy which don't like ambiguous base pairs.

### Usage
``` 
python reambap.py [input] [output]
```
